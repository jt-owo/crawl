# crawl

## Controls

### Modes

- <kbd>I</kbd>    = Game Info
- <kbd>`</kbd>    = Movement Mode
- <kbd>L</kbd>    = Selector Mode
- <kbd>F12</kbd>  = Quit

### Movement

- <kbd>W</kbd>    = Up
- <kbd>A</kbd>    = Left
- <kbd>S</kbd>    = Down
- <kbd>D</kbd>    = Right

### Interaction

- <kbd><</kbd> Go up stairs
- <kbd>></kbd> Go down stairs
